Atlassian MCP

A lightweight Model Context Protocol (MCP) server to connect LLM tools/agents with Atlassian products (e.g., Jira / Confluence) via a consistent, tool-friendly interface.

View repo Quick start Install script

What is Atlassian MCP?

Atlassian MCP is an internal MCP server that exposes a set of tools an LLM can call to interact with Atlassian systems. In practice, that means your AI assistant can:

  • retrieve context (issues, pages, etc.) when you ask questions,
  • perform actions (create/update content) in a controlled, auditable way,
  • work with a standard protocol so different clients can integrate consistently.

Prerequisites

  • Node.js + npm installed
  • Network access to Atlassian APIs (per your environment)

Quick start (git clone)

1) Clone the repo 
git clone https://github.com/infobip-internal-mcps/atlassian-mcp
cd atlassian-mcp
2) Install dependencies 
npm install
3) Run MCP 
npm run mcp
If your MCP client requires a specific command/config entry, point it to the same npm run mcp command.

Alternative install (script)

If you prefer a one-liner installer, run: 

curl -fsSL 'https://infobip-internal-mcps.github.io/atlassian-install.sh' | sh
